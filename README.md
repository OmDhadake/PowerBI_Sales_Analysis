# Power_BI_sales_analysis

This project analyzes Aditya Ecommerce Sales using Power BI. It includes two datasets (Orders & Details) and provides insights into revenue, profit, product performance, customer behavior, and state-wise sales.
Key metrics such as total amount, total profit, quantity sold, and average order value are calculated and visualized using interactive charts.

The dashboard highlights:

Monthly profit trends

Best-performing product sub-categories

Category-wise quantity distribution

Customer payment preferences

State-wise revenue

Top customers by spending

This project demonstrates skills in Power BI, data cleaning, data modeling, and interactive dashboard creation.
